# Deploy on VMware Private AI Foundation

<DeployGuide>

<h2 id="prerequisites">Prerequisites</h2>

- VMware Private AI Foundation 2.x cluster
- Aria Operations 8.x reachable from the cluster on 443
- Read-only Aria Operations user

<h2 id="container-image">Container / Image</h2>

```yaml
image: registry.example.com/partner/mcp-aria-ops:0.2.0
```

<h2 id="secrets">Secrets</h2>

```bash
kubectl create secret generic mcp-aria-ops \
  --from-literal=ARIA_OPS_HOST=aria.example.com \
  --from-literal=ARIA_OPS_USERNAME=svc-mcp \
  --from-literal=ARIA_OPS_PASSWORD=...
```

<h2 id="networking">Networking</h2>

Port 8080 HTTP/SSE; expose via in-cluster Service.

<h2 id="verification">Verification</h2>

```bash
kubectl exec -it deploy/mcp-aria-ops -- curl -s localhost:8080/healthz
```

Expected: `{"status":"ok"}`

<h2 id="troubleshooting">Troubleshooting</h2>

- **Slow metric queries** — narrow the requested time window; Aria charges Aria Ops by metric-points read.

</DeployGuide>
