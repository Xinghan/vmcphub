Prerequisites

  • VMware Cloud Foundation 2.x cluster
  • Outbound HTTPS to your Jira Cloud or Data Center instance
  • A Jira API token (Cloud) or PAT (DC)

Container / Image

image: registry.example.com/community/mcp-jira:0.3.0

Secrets

kubectl create secret generic mcp-jira \
  --from-literal=JIRA_BASE_URL=https://example.atlassian.net \
  --from-literal=JIRA_API_TOKEN=...

Networking

If your VCF cluster egresses through a proxy, set HTTPS_PROXY and NO_PROXY on the Deployment so MCP traffic stays inside the cluster while Jira requests use the proxy.

Verification

kubectl exec -it deploy/mcp-jira -- curl -s localhost:8080/healthz

Expected: {"status":"ok"}

Troubleshooting

  • 401 from Jira Cloud — verify the token is scoped to the project you query.
  • JQL parse errors — escape reserved characters in issue keys when used in summary ~ "...".
