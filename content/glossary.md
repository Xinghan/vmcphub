# Glossary

- **MCP** — Model Context Protocol. A protocol that lets LLM clients call tools
  and read resources from an external server.
- **Skill** — A bundled set of prompts, tools, and data that an assistant can
  invoke by name to perform a specific task.
- **VCF** — VMware Cloud Foundation, VMware's stack for running LLM
  workloads on-prem.
- **Tier** — Trust level of an entry: Official (VMware), Verified
  (third-party reviewed by VMware), or Community (popular tools with VMware
  deployment guidance).
