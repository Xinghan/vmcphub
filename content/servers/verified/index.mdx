**Third-party (mostly open-source) MCP servers that target VMware products** — vCenter, NSX, vSAN, VMware Cloud Foundation, and friends. These are not authored by VMware, but VMware has reviewed each one and confirmed it works correctly against a recent VMware product release. Use these when an Official server doesn't exist yet, or when you prefer the upstream community implementation.
