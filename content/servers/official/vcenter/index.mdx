The vCenter MCP Server exposes vSphere inventory, VM lifecycle operations, host
and cluster status, and read-only performance counters as MCP tools and
resources. Use it to let an AI assistant answer questions about your vSphere
estate or perform routine VM operations under tight permission scopes.

## Prerequisites

- vCenter 7.0 Update 3 or later
- A service account with the minimum role described in the repo README
- Network reachability from the MCP client (or PAIF runtime) to vCenter on 443
