After pasting the snippet into your client's MCP config, set the following
environment variables (or use a `.env` file in your MCP runtime):

- `VCENTER_HOST` — e.g. `vcsa.example.com`
- `VCENTER_USERNAME` — service account UPN
- `VCENTER_PASSWORD` — service account password
