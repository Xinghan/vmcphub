# Deploy on VMware Cloud Foundation

<DeployGuide>

<h2 id="prerequisites">Prerequisites</h2>

- VMware Cloud Foundation 2.x cluster
- NSX MCP Server deployed in the same namespace

<h2 id="container-image">Container / Image</h2>

Not applicable — skill bundle distributed as ConfigMap or OCI artifact.

<h2 id="secrets">Secrets</h2>

No skill-specific secrets.

<h2 id="networking">Networking</h2>

No network listeners.

<h2 id="verification">Verification</h2>

Invoke the skill in VCF chat and confirm the policy template prints.

<h2 id="troubleshooting">Troubleshooting</h2>

- **Skill suggests rules the NSX MCP server rejects** — confirm the NSX user has `auditor` role for read validation.
