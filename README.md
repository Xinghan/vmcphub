# vmcphub

VMware MCP Server & Skills portal hub. Built with Docusaurus 3.

A static catalog of MCP servers and Skills relevant to VMware Private AI Foundation users, organized by trust tier (Official / Verified / Community) and capability.

## Development

```bash
pnpm install
pnpm start
```

## Build

```bash
pnpm build
pnpm serve
```

See `docs/superpowers/specs/2026-05-25-vmcphub-design.md` for the design and `docs/superpowers/plans/2026-05-25-vmcphub.md` for the implementation plan.
